L'utilisateur de EASYSCHOOLREPORT a le libre choix d'initialiser le système dans son ordre voulu. Néanmoins, pour éviter certains embrouilles, le Concepteur du système propose l'ordre suivant :


>>étape 0 : Créer des comptes d'utilisateur.

>>étape 1 : Entrer les propriétés de l'établissement.

>>étape 2 : Entrer tous les Départements(Spécialité) de l'établissement.

>>étape 3 : Entrer toutes les classes de l'établissement.

>>étape 4 : Entrer toutes les matières de l'établissement.

>>étape 5 : Atttribuer les matières aux classes, en précisant les groupes et les coefficients.

>>étape 6 : Entrer les élèves dans leurs différentes classes.

>>étape 7 : Entrer les enseignants(Personnels).

>>étape 8 : Entrer les mots de passe des enseignants

>>étape 9 : Attribuer les enseignants aux classes

>>étape 10 : Attribuer les professeurs principaux

>>étape 11 : Indiquer les marges pour avecrtissements et blâmes

>>étape 12 : Attribuer des photos aux élèves

>>étape 13 : Ainsi de suite... jusqu'à vous rassurer que vous avez tout paramétré...
